// how it works
Step 01
Structure detection
The file is split on --- separators into sections. Each section's frontmatter is parsed for card_type and title.
card_type: cue_card
title: Introduction to NumPy
---
content here…
Step 02
Cell classification
A line-by-line state machine classifies each code fence. Python fences become code cells. Bash/output fences become expected-output blocks.
```python → code cell
```bash → output block
```output → output block
plain text → markdown cell
Step 03
Notebook assembly
Cells are assembled into nbformat 4 JSON with Colab metadata, a Python 3 kernelspec, and unique cell IDs. The .ipynb downloads instantly.
{
"nbformat": 4,
"metadata": {
"colab": {…}
},
"cells": […]
}
// supported formats
Code cell fences
  • ```python Standard Python fence
  • ```python! Runnable variant
  • ```python= Display variant
  • ```py Short alias
  • ```python3 Explicit Python 3
Output / markdown fences
  • ```bash Becomes expected output block
  • ```output Direct output annotation
  • ```text Plaintext output
  • ```shell Shell output
  • ```console Console output
Card types
  • cue_card Converted → cells
  • quiz_card Skipped (instructor-only)
Stripped content
  • [Note to Instructor] Span + blockquote removed
  • [Ask Learners] Span + blockquote removed
